Bigfoot sightings

Data maps are very popular as more and more spatial datasets are available. The following figure displays Bigfoot sightings for the 50 states.

# Attach data
attach(bigfoot)

# Load packages
# You need to do this everytime you are using a 
# function from a specific package
library(ggplot2)
library(fiftystater)

# map_id creates the aesthetic mapping to the state name column in your data
p <- ggplot(bigfoot, aes(map_id = tolower(State))) +
  
  # map points to the fifty_states shape data
  geom_map(aes(fill = Bigfoot), map = fifty_states) +
  expand_limits(x = fifty_states$long, y =  fifty_states$lat) +
  coord_map() + labs(title="Bigfoot Sightings") +
  scale_fill_gradientn(colours=rev(heat.colors(10))) +
  scale_x_continuous(breaks = NULL) +
  scale_y_continuous(breaks = NULL) +
  labs(x = "", y = "") +
  theme(legend.position = "bottom",
        panel.background = element_blank())

# add border boxes to AK/HI
p + fifty_states_inset_boxes()

The following figure displays a bivariate plot combining Bigfoot sightings and population in millions for the 50 states.

## Bivariate plot

# Load package
library(colorplaner)

p + aes(fill2 = Pop.m) + scale_fill_colorplane() +
  theme(legend.position = "right")
